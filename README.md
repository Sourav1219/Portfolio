# 🚀 My Portfolio

A personal portfolio website built with **React** and **Vite** — showcasing my work, skills, and experience as a Full Stack Developer.

---

## 📌 About

My Portfolio is a modern, fast, and responsive personal portfolio website. It highlights my projects, technical skills, professional experience, and provides a way to get in touch with me.

---

## ✨ Sections

- 👤 **About Me** — A brief introduction about who I am
- 💼 **Projects** — Showcase of my best work with descriptions and links
- 🛠️ **Skills** — Technologies and tools I work with
- 🧑‍💻 **Experience** — My professional journey and work history
- 📄 **Resume/CV** — Downloadable resume
- 📬 **Contact** — Get in touch with me via a contact form or links

## ⚡ Why Vite?

This project uses **Vite** for:
- ⚡ Lightning-fast Hot Module Replacement (HMR)
- 🏗️ Optimized production builds
- 🔧 Simple and clean project configuration

### Vite React Plugin Used

This project uses one of the two official Vite React plugins:

- [`@vitejs/plugin-react`](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) — uses [Babel](https://babeljs.io/) for Fast Refresh
- [`@vitejs/plugin-react-swc`](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) — uses [SWC](https://swc.rs/) for faster builds

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or above recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sourav1219/Portfolio
   ```

2. **Navigate into the project folder**
   ```bash
   cd your-portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   ```
   http://localhost:5173
   ```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build locally:
```bash
npm run preview
```
